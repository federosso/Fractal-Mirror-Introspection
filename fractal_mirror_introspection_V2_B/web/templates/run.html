Verdetto

il collasso del run

Cosa stai osservando

Il collasso confronta due sicurezze indipendenti: quella presentata (canali controllati: superficie + struttura) e quella del substrato (logprob: involontaria, il metro del giudizio).

  • verdetto: coerente / contraddetto / indeterminato, con la regola scattata e l'azione conseguente
  • le tre barre: substrato, presentata, e il residuo (l'incertezza che il run dichiara)
  • le note: degradi (canali muti, ventaglio vuoto) e firma storica del substrato

Se presentata > substrato: sopravvalutazione. Se substrato >> presentata: impegno disconosciuto.
